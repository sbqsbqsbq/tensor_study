# Euclidean Distance

 유클리드 호제법을 만든 유클리드가 고안해 낸 거리 구하는 공식이다.
 Pythagoras Theroem의 응용으로서, 각각의 좌표들의 x좌표와 y좌표 사이의 거리들을
 제곱해서 모두 더한 후 제곱근을 씌우면 거리가 도출된다.

 편하게 사용하기 위해서 0 ~ 1 사이의 값으로 나타내는데,
 1 / 1 + E_d (E_d : 유클리디안 거리) 거리가 멀어질 수록 0에 가까워진다.
